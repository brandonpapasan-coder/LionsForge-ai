# LionsForge AI Market Simulator Execution Engine

## Purpose
Define the first executable logic layer for the educational paper-trading simulator.

## Execution Flow

1. Receive simulated order request
2. Validate account state
3. Validate available virtual capital or position size
4. Generate simulated fill
5. Update portfolio state
6. Record trade event
7. Calculate performance metrics
8. Generate educational feedback

## MVP Services

### Order Simulation Service
- Buy simulation
- Sell simulation
- Position updates
- Transaction history

### Portfolio Calculation Service
- Market value calculation
- Profit/loss calculation
- Allocation percentages
- Exposure tracking

### Risk Calculation Service
- Position concentration
- Portfolio volatility estimate
- Drawdown tracking
- Risk education alerts
