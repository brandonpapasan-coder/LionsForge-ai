# LionsForge AI Market Simulator MVP

## Purpose
Educational paper-trading simulation and research validation environment.

## Guardrails
- No live trading execution
- No brokerage integration
- No real-money transactions

## Core Components

### Simulation Account
Tracks:
- virtual cash
- positions
- portfolio value
- performance history

### Virtual Trade Engine
Supports:
- simulated buy orders
- simulated sell orders
- transaction history
- cost basis tracking

### Scenario Engine
Initial scenarios:
- bull market
- bear market
- volatility shock
- inflation environment
- interest rate changes

### Strategy Lab
Evaluates:
- returns
- volatility
- drawdown
- diversification
- risk exposure

### AI Mentor
Provides:
- strategy explanations
- decision feedback
- educational recommendations

## Integration Targets
- Research Trust Index
- Validation Engine
- Education Hub
- Knowledge Memory

## MVP Build Order
1. Database models
2. Simulation APIs
3. Calculation engine
4. Scenario replay
5. Education feedback layer
6. Automated tests
