# Security Policy

## Supported project status

LionsForge AI is in early MVP development. Security fixes should be prioritized before production launch.

## Reporting a vulnerability

Do not open public issues for sensitive vulnerabilities.

Report security concerns privately to the repository owner.

## Secret handling

- Do not commit real API keys.
- Do not commit production database credentials.
- Use environment variables for secrets.
- Rotate any secret that is accidentally committed.

## Financial application note

Features involving live trading, portfolio monitoring, or user financial data require additional review before production use.
